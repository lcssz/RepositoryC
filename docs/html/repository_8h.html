◆ process_music_file_for_word_count()

void process_music_file_for_word_count ( const char * filepath)

Processa arquivo de música para contagem de palavras (versão básica)

Parâmetros
filepathCaminho para o arquivo de música
filepathCaminho para o arquivo de música

Lê um arquivo .txt contendo uma letra de música e processa cada linha, inserindo as palavras nas estruturas básicas (árvores binária e AVL). Esta é uma versão simplificada que não armazena informações completas como título, autor e trechos da música.

Nota
Esta função assume que o arquivo tem o formato:
  • Primeira linha: título da música
  • Segunda linha: autor
  • Linhas subsequentes: letra da música
Aviso
A função não valida extensivamente o formato do arquivo e pode falhar silenciosamente se o arquivo não existir ou estiver mal formatado.

◆ process_line_and_insert_words()

void process_line_and_insert_words ( const char * line)

Processa uma linha e insere palavras nas estruturas básicas.

Parâmetros
lineLinha a ser processada
lineLinha a ser processada

Divide a linha em tokens usando strtok() e insere cada palavra válida (com 3 ou mais caracteres) nas árvores binária e AVL básicas. Esta função é usada na versão simplificada do processamento.

Aviso
Esta função termina o programa (exit) se houver falha na alocação de memória, pois é uma versão simplificada para demonstração.
Nota
As palavras são inseridas diretamente sem limpeza adicional, assumindo que já foram processadas adequadamente.

◆ process_music_file_complete()

void process_music_file_complete ( const char * filepath,
Repository * repo )

Processa arquivo de música completo com todas as informações.

Parâmetros
filepathCaminho para o arquivo de música
repoRepositório onde armazenar os dados
filepathCaminho para o arquivo de música
repoRepositório onde armazenar os dados

Esta é a função principal de processamento que lê um arquivo .txt contendo uma letra de música e processa cada palavra, inserindo-as nas três estruturas de dados do repositório (vetor, árvore binária e árvore AVL) com informações completas.

A função:

  • Lê o título e autor da música
  • Processa cada linha da letra
  • Limpa cada palavra (remove pontuação, converte para minúsculo)
  • Filtra palavras com menos de 4 caracteres
  • Cria estruturas WordInfo com informações completas
  • Insere nas três estruturas de dados
Nota
Formato esperado do arquivo:
  • Linha 1: Título da música
  • Linha 2: Autor da música
  • Linhas 3+: Letra da música
Complexidade: O(n * log m) onde n é o número de palavras e m é o tamanho das estruturas de dados.
Aviso
A função assume que o arquivo está bem formatado e pode falhar silenciosamente se o arquivo não existir ou estiver mal formatado.

void search_word_in_all_structures ( Repository * repo,
const char * word )

Busca uma palavra em todas as estruturas de dados.

Parâmetros
repoRepositório onde buscar
wordPalavra a ser buscada
repoRepositório onde buscar
wordPalavra a ser buscada

Realiza a busca da palavra especificada em todas as três estruturas de dados (vetor, árvore binária e árvore AVL) e mede o tempo de execução de cada busca. Os resultados são exibidos com os tempos de performance para comparação.

A função:

  • Limpa a palavra de entrada (remove pontuação, converte para minúsculo)
  • Executa busca no vetor usando busca binária
  • Executa busca na árvore binária de busca
  • Executa busca na árvore AVL
  • Mede o tempo de cada operação usando clock()
  • Exibe os resultados formatados
Nota
Complexidade das buscas:
  • Vetor (busca binária): O(log n)
  • Árvore Binária: O(log n) no melhor caso, O(n) no pior caso
  • Árvore AVL: O(log n) garantido
A função assume que o vetor está ordenado para busca binária. Se não estiver ordenado, a busca no vetor pode retornar resultados incorretos.
Aviso
A função não valida se o repositório foi inicializado corretamente ou se as estruturas de dados estão em um estado válido.
