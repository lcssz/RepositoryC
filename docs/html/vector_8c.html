Descrição detalhada

Implementação do vetor dinâmico com busca binária.

Autor
AED2 - Algoritmos e Estruturas de Dados II
Versão
1.0

Este arquivo contém a implementação de um vetor dinâmico que armazena estruturas WordInfo e suporta busca binária após ordenação. O vetor pode ser expandido dinamicamente conforme necessário.
